# App Docs

This folder is for documentation about Nucleus itself: how to build it, run it, ship it, and use its app-specific workflows.

## What Belongs Here

- setup and development instructions
- release and distribution steps
- Nucleus feature docs
- integration details for MEX and local tooling

## Current Docs

- [DEVELOPMENT.md](DEVELOPMENT.md) - local development setup and project layout
- [README_MEX.md](README_MEX.md) - MEX project workflow and usage notes
- [MEX_INTEGRATION.md](MEX_INTEGRATION.md) - implementation details for the MEX integration
- [ROADMAP.md](ROADMAP.md) - planned and historical feature tracking
- [DISTRIBUTION_GUIDE.md](DISTRIBUTION_GUIDE.md) - packaging and release notes
- [LINUX_BUILD.md](LINUX_BUILD.md) - Linux-specific build notes

## What Does Not Belong Here

General Melee modding knowledge, file format notes, reverse-engineering notes, and research writeups belong in the [wiki](../wiki/README.md).

## Legacy Research Material

The existing [color-effects-reference](color-effects-reference/) folder is still useful, but it is raw research material rather than polished app documentation. As that knowledge gets cleaned up, it should move into the wiki as curated pages.
